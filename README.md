# English Pronunciation Evaluator 🎙️

An elegant, simple, and modular Streamlit web application designed for students and English language learners to assess their spoken English pronunciation and fluency. 

The application utilizes **faster-whisper** for local audio transcription and word-level confidence scores, a local **HuggingFace** transformer model to suggest grammar-context alternatives, and the **OpenAI API** (with a robust offline fallback) to generate articulatory feedback.

---

## Features

- **Dual Input Modes**: Upload audio files (`WAV`, `MP3`, `M4A`) or record audio live using the built-in browser microphone widget.
- **Audio Validation**: Automatically enforces duration limits (strictly **30 to 45 seconds**) using `ffprobe` to ensure standard compliance.
- **Heuristic Pronunciation Scoring**: Evaluates speech and awards a final score out of 100 based on word confidence, speech pauses/hesitations, and acoustic noise.
- **Visual Highlighter**: Displays the transcribed text where mispronounced words (red, dashed underline) and filler words (orange, dotted underline) are highlighted with interactive tooltips.
- **Integrated AI & Local Coaching**: Displays feedback containing structural pronunciation advice alongside grammatical synonyms generated by a local HuggingFace `fill-mask` pipeline.
- **Stateless & Privacy-Compliant**: Fully aligned with India's DPDP regulations. It requires user consent and operates with a zero-retention file policy (files are deleted immediately after transcription).

## Installation & Setup

### Prerequisites
Make sure you have **Python 3.8 - 3.12** and **FFmpeg** installed on your system path.

1. **Clone or Extract the Workspace**:
   ```bash
   cd pronunciation-app
   ```

2. **Set Up a Virtual Environment**:
   ```bash
   python -m venv .venv
   ```

3. **Activate the Virtual Environment**:
   - **Windows (PowerShell)**:
     ```powershell
     .venv\Scripts\Activate.ps1
     ```
   - **Windows (CMD)**:
     ```cmd
     .venv\Scripts\activate.bat
     ```
   - **macOS / Linux**:
     ```bash
     source .venv/bin/activate
     ```

4. **Install Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

---

## Running Locally

1. Set your OpenAI API key in your terminal session, or input it directly into the application sidebar:
   ```bash
   # Windows (PowerShell)
   $env:OPENAI_API_KEY="your-api-key-here"

   # macOS / Linux
   export OPENAI_API_KEY="your-api-key-here"
   ```

2. Run the Streamlit application:
   ```bash
   streamlit run app.py
   ```

3. Open your browser and navigate to the local address displayed in the terminal (usually `http://localhost:8501`).

## Cloud Deployment

### 1. Deploying to Streamlit Community Cloud
Deploying is simple since there is no database:
1. Push this project folder to a public GitHub repository.
2. Visit [Streamlit Share](https://share.streamlit.io/) and log in with your GitHub account.
3. Click **New app**, select your repository, branch, and set the entrypoint file to `app.py`.
4. In **Advanced Settings**, add your `OPENAI_API_KEY` under the Secrets text box.
5. Click **Deploy**. Streamlit Cloud automatically installs packages in `requirements.txt` and starts the server.

### 2. Deploying to Render
1. Create a new **Web Service** on Render and link your repository.
2. Use the **Python** environment.
3. Set the **Build Command**:
   ```bash
   pip install -r requirements.txt
   ```
4. Set the **Start Command**:
   ```bash
   streamlit run app.py --server.port $PORT --server.address 0.0.0.0
   ```
5. In **Environment Variables**, add `OPENAI_API_KEY` if desired.
